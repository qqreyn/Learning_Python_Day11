# Learning_Python_Day11

Today's topic is a full project!

**Topic:** Project – In-Memory User System  

**Goal:** Combine lists and dictionaries to build a functional user management system

**Steps:**

1. Create a list that will store multiple user dictionaries  
2. Add a few sample users (each user should have username, email, age, etc.)  
3. Write a function to find a user by username  
4. Implement a function to check if a username already exists  
5. Create a function to validate user data (make sure fields are not empty)  
6. Build a function to display all users in a formatted way  
7. Connect everything into one working console program
